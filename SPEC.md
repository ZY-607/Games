# 武侠幸存者：江湖之路 游戏规格文档

## 1. 项目概述

### 项目名称
**武侠幸存者：江湖之路**

### 项目类型
HTML5 Canvas 动作肉鸽游戏（Survivor Roguelike）

### 核心功能概述
玩家扮演一名江湖侠客，在无尽的怪物浪潮中生存。通过击败敌人获得经验和武学秘籍，构筑独特的build系统，不断提升战力对抗越来越强大的敌人。

### 目标用户
- 喜欢肉鸽/幸存者类游戏的玩家
- 武侠题材爱好者
- 喜欢太吾绘卷美术风格的玩家

---

## 2. UI/UX 规格

### 2.1 布局结构

**游戏画布区域**
- 全屏Canvas渲染
- 分辨率自适应：1920x1080 基准，保持16:9比例
- 最小支持：1280x720

**UI层级顺序**（从底到顶）
1. 游戏背景层（静态/动态）
2. 游戏实体层（角色、敌人、掉落物）
3. 特效层（粒子、伤害数字）
4. UI信息层（血条、经验条）
5. 交互UI层（升级面板、暂停菜单）
6. 顶级UI层（设置、全屏提示）

### 2.2 视觉设计

**色彩方案**
- 主色调：#2D1B0E（深棕，江湖古风）
- 次色调：#8B4513（棕色，木质元素）
- 强调色：#D4AF37（金色，武学光芒）
- 背景色：#1A1A2E（深蓝黑，夜色江湖）
- 血量色：#E74C3C（红色）
- 经验色：#F39C12（橙黄）
- 内力色：#3498DB（蓝色）
- 毒属性：#27AE60（绿色）
- 火属性：#E67E22（橙红）
- 冰属性：#74B9FF（浅蓝）

**字体方案**
- 主标题：'ZCOOL KuaiLe' 或 'Ma Shan Zheng'（书法风格）
- 正文/UI：'Noto Sans SC' 或 'Microsoft YaHei'
- 数字/伤害：'Press Start 2P'（像素风格）

**字号规格**
- 标题：48px
- 副标题：32px
- 正文：18px
- 小字：14px
- 伤害数字：24-36px

**间距系统**
- 基础单位：8px
- 组件内边距：16px
- 组件间距：24px
- 面板边距：32px

### 2.3 组件设计

**角色属性面板**（左上角）
- 背景：半透明深棕（rgba(45,27,14,0.85)）
- 边框：2px金色描边
- 圆角：8px
- 显示内容：
  - 角色名称/境界
  - 生命值条（当前/最大）
  - 攻击力
  - 攻击速度
  - 移动速度
  - 防御力
  - 暴击率
  - 暴击伤害
  - 闪避率
  - 各种抗性

**技能面板**（右上角）
- 显示已装备的外功和内功
- 图标+等级显示
- 点击可查看详情

**升级选择面板**（中央弹出）
- 三个技能卡牌横向排列
- 每张卡片显示：技能图标、名称、描述、当前等级
- 相同技能叠加时显示升级效果
- 卡片悬停放大效果

**敌人信息**（血条显示）
- 位于敌人头顶
- 精英怪金色边框
- BOSS红色边框+名称

**暂停菜单**
- 半透明黑色遮罩
- 中央菜单框
- 选项：继续游戏、重新开始、退出游戏

### 2.4 动画效果

**角色动画**
- 移动：像素行走动画（4帧循环）
- 攻击：攻击动作帧
- 受击：闪烁效果

**粒子特效**
- 击杀敌人：经验球粒子
- 暴击：金色星星
- 技能特效：各技能独特效果

**伤害数字**
- 飘字动画：向上飘动+淡出
- 普通伤害：白色
- 暴击伤害：金色放大
- 治疗效果：绿色

## 3. 功能规格

### 3.1 角色系统

**基础属性**
- 生命值：100（可提升）
- 攻击力：10（可提升）
- 攻击速度：1.0次/秒（可提升）
- 移动速度：3（可提升）
- 防御力：0（可提升）
- 暴击率：5%（可提升）
- 暴击伤害：150%（可提升）
- 闪避率：0%（可提升）
- 抗性：各属性抗性初始0%（可提升）

**境界系统**
- 炼体期（1-9级）
- 筑基期（10-19级）
- 金丹期（20-29级）
- 元婴期（30-39级）
- 化神期（40-49级）
- 渡劫期（50级以上）

| 神照经 | 生命低于30%时触发无敌 | 无敌持续+1秒 |

### 3.4 敌人系统

**普通敌人**
- 喽啰：基础敌人，弱
- 恶丐：群体攻击
- 杀手：远程投射
- 悍匪：高攻低防

**精英敌人**
- 金刚： 高防御
- 毒魔： 中毒攻击
- 鬼魅： 高闪避
- 炎魔： 火焰攻击

**BOSS敌人**
- 第一关：血刀老祖
- 第二关：欧阳锋
- 第三关：扫地僧
- 第四关：东方不败
- 第五关及以后：逍遥子（循环）

**敌人属性成长**
- 生命：每波+10%
- 攻击力：每波+5%
- 数量：每波+2

### 3.5 波次系统

- 每波持续时间：60秒
- 间隔时间：5秒（显示"下一波即将来袭"）
- 波次奖励：武学宝箱
- 难度递增：线性增长
- 精英刷新：第3波开始，每波+1精英
- BOSS：每5波出现

### 3.6 掉落系统

- 经验球：击杀敌人掉落
- 金币：稀有掉落，用于抽取
- 宝箱：波次奖励
- 磁铁效果：自动吸附周围掉落物

### 3.7 交互设计

**操作方式**
- WASD：移动
- ESC：暂停
- 鼠标点击：与UI交互

**自动攻击**
- 角色自动攻击范围内最近的敌人
- 优先攻击低血量敌人
