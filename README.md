# 武侠幸存者：江湖之路

## 项目概述

《武侠幸存者：江湖之路》是一款基于HTML5 Canvas开发的动作肉鸽游戏（Survivor Roguelike），玩家扮演一名江湖侠客，在无尽的怪物浪潮中生存，通过击败敌人获得经验和武学秘籍，构筑独特的build系统，不断提升战力对抗越来越强大的敌人。

## 开发进度

### ✅ 已完成功能

#### 核心系统
- **玩家系统**：WASD移动控制，自动攻击机制，属性成长系统
- **角色选择系统**：4个可选角色，各有独特属性和被动技能
- **地图系统**：程序化随机地图生成，瓦片碰撞检测，森林风格
- **敌人系统**：普通敌人、精英敌人、BOSS敌人，波次递增系统
- **技能系统**：20种外功技能（主动），20种内功技能（被动）
- **技能联动系统**：技能协同组合、元素词缀叠加、流派倾向加成
- **升级系统**：经验获取，等级提升，三选一技能升级面板
- **波次系统**：每波60秒，间隔5秒，难度逐渐递增
- **掉落系统**：经验球掉落，磁铁吸附功能
- **金币系统**：怪物掉落金币，小怪随机掉落，精英/BOSS必掉
- **晶石系统**：仅BOSS类敌人掉落晶石，用于天赋树解锁
- **局外成长系统**：永久属性加成，武馆修炼商店
- **角色天赋树系统**：每个角色独立网状天赋，金币+晶石双资源解锁
- **最终BOSS系统**：10分钟触发最终BOSS，胜利结算
