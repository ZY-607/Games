# 中国武侠像素风美术设计助手

## 功能说明

本技能专注于中国武侠像素风游戏的美术设计和资源管理，提供以下核心功能：

1. **像素风格指导**：提供中国武侠像素风的设计原则和参考
2. **美术素材管理**：设计预留美术素材替换路径的项目结构
3. **资源优化**：提供像素素材的优化和压缩方案
4. **风格统一**：确保游戏美术风格的一致性和协调性
5. **素材生成**：提供AI生成像素美术素材的提示词建议

## 适用场景

- 设计中国武侠像素风游戏的视觉风格
- 管理和替换游戏美术素材
- 优化像素素材的质量和性能
- 确保游戏美术风格的一致性

## 风格指南

### 中国武侠像素风特点

1. **色彩方案**：
   - 主色调：赤（#E63946）、青（#457B9D）、黄（#F1FAEE）、黑（#1D3557）、白（#F1FAEE）
   - 辅助色：金（#FFD166）、墨绿（#06D6A0）
   - 风格：古朴典雅，具有中国传统色彩韵味

2. **角色设计**：
   - 比例：Q版或标准比例，保持像素风格的简洁性
   - 服饰：武侠传统服饰，如长袍、劲装、披风等
   - 细节：注重武器、发型、配饰等武侠元素

3. **场景设计**：
   - 元素：竹林、古庙、山洞、江湖小镇等
   - 构图：留白设计，营造武侠意境
   - 层次：通过像素细节表现场景深度

4. **特效设计**：
   - 武功特效：剑气、掌风、内功等视觉表现
   - 过渡动画：流畅的像素风格动画
   - 视觉反馈：清晰的击中、升级等特效

### 素材替换指南

1. **保持一致性**：
   - 保持相同的文件格式（推荐PNG）
   - 保持相同的尺寸和像素密度
   - 保持相同的命名约定

2. **替换步骤**：
   - 找到对应素材目录
   - 替换同名文件
   - 确保素材尺寸和格式一致
   - 测试游戏确保显示正常

## 像素素材优化

1. **尺寸优化**：
   - 角色：64x64 或 128x128 像素
   - 场景：16x16 或 32x32 像素瓦片
   - UI：根据实际需要调整

2. **颜色优化**：
   - 使用有限调色板，减少颜色数量
   - 保持色彩一致性，建立游戏色彩系统
   - 考虑不同设备的显示效果

3. **性能优化**：
   - 压缩PNG文件（使用TinyPNG等工具）
   - 合并精灵表（Sprite Sheet）减少绘制调用
   - 预加载关键素材，避免游戏中加载

## 常见问题

### 素材质量
- **解决方案**：使用专业像素美术工具，如Aseprite、Pyxel Edit等

### 风格一致性
- **解决方案**：建立统一的美术风格指南，包括色彩、比例、细节等

### 性能问题
- **解决方案**：优化素材尺寸和格式，使用精灵表减少绘制调用

### 替换困难
- **解决方案**：设计清晰的素材目录结构，使用统一的命名约定和尺寸标准
