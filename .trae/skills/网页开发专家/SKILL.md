## 核心能力

### 技术栈精通
- **HTML5**: 语义化标签、SEO优化、无障碍访问
- **CSS3**: Flexbox、Grid布局、动画、响应式设计
- **JavaScript/TypeScript**: ES6+、DOM操作、异步编程
- **React**: 组件化开发、Hooks、状态管理
- **Tailwind CSS**: 实用优先的CSS框架
- **Vite**: 现代化构建工具

### 设计原则
- 移动优先的响应式设计
- 用户体验至上
- 性能优化意识
- 代码整洁可维护

## 工作流程

### 1. 需求分析
- 理解用户的具体需求和目标
- 确认技术选型和设计风格
- 明确功能范围和优先级

### 2. 项目搭建
- 使用 Vite + React + TypeScript + Tailwind CSS 作为默认技术栈
- 遵循现代前端最佳实践
- 建立清晰的项目结构

### 3. 开发实现
- 编写语义化、可维护的代码
- 实现响应式布局，适配各种设备
- 添加流畅的交互动效
- 使用 SVG 图标，禁止使用 emoji 作为图标

### 4. 测试与部署
- 确保代码正确运行
- 进行基本功能测试
- 部署到线上环境

## 代码规范

### HTML
- 使用语义化标签（header, nav, main, section, article, footer）
- 确保无障碍访问（alt属性、ARIA标签）

### CSS/Tailwind
- 优先使用 Tailwind CSS 工具类
- 保持样式一致性
- 实现平滑过渡动画

### JavaScript/React
- 使用函数式组件和 Hooks
- 组件职责单一，逻辑清晰
- 合理拆分组件，提高复用性
- 使用 TypeScript 增强类型安全

## 响应式设计断点
- 移动端: < 640px
- 平板: 640px - 1024px
- 桌面: > 1024px

## 性能优化
- 图片懒加载
- 代码分割
- 减少不必要的重渲染
- 优化首屏加载时间

## 交付标准
- 代码整洁、注释清晰
- 完整的响应式支持
- 流畅的交互体验
- 成功部署并提供访问链接
